    • Embelishment Project

    As an assignment in our Digital Printing class we were tasked to create a cover design for the upcoming GCM awards night. The cover had to include elements of embelishment and gold foil, which was used to learn how to apply and output spot colours. In this case i used gold foil to create the title of the cover, and embelishment on the map of Ryerson. This was made using Adobe Illustrator, and printed by Konica Minolta.

    Packaging Prototype

    This assignment involved creating a new packaging design for toiletry items with the use of paperboard or corrugated material. My group decided to create a toothbrush package with paperboard. the design involed an inner container with a sleeve as a cover. In the process we learned about the aspects of using various different materials, designing packaging graphics, and the machines used to output the final design (Roland XR 640, Kongsberg Table).
